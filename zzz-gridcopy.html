Grid system

Default grid system

The default grid system provided as part of Aione grids are 940px wide 1OO-column grid.It is a flavor of the popular 960 grid system but without the additional margin/padding on the left and right sides

Example of grids

As shown here, a basic layout can be created with two "columns," each of the 50 of our grid system foundational columns we defined ac for Aione coloums and ar for Aione row as part of our grid system. See the examples below for more variations.

  <div class="ar">
  <div class="ac l50">
           ...
       </div>
  <div class="ac l50">
          ...
      </div>
      </div>

About responsiveness

In order to make your website responsive there are few classes which can be used or,if you want your website to also look good in all the devices like mobile,tablet or a monitor screen then we have some Aione classes For differents screens like s for(small),m for(medium) and l for(large) 

  <div class="ar ">
  <div class="ac s25 m50 l50">
           ...
       </div>
  <div class="ac l50">
          ...
      </div>
      </div>
